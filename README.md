# Daily Expense Tracker (Table Format)

A minimalist, table-centric expense tracker for daily financial logging.

## Features
- **Side-by-Side Columns**: Date, Earns, Other Income, Spends, and Balance.
- **Auto-Calculation**: Balance updates instantly (Earns + Other Income - Spends).
- **Persistence**: Data is automatically saved to your browser's local storage.
- **Dynamic Rows**: Add or delete daily entries with a single click.
- **Visual Feedback**: Positive balances are shown in green, negative in red.

## Columns
1. **Date**: Select the day for the entry.
2. **Earns (₹)**: Main daily earnings.
3. **Other Income (₹)**: Any auxiliary income.
4. **Spends (₹)**: Daily expenditure.
5. **Balance (₹)**: Calculated total for the day.

## How to use
- Open `index.html` in any browser.
- Use the **+ Add New Day** button to start a new entry.
- All totals are summarized at the bottom of the table.
